jvfe's SPARQL queries

Some of my useful Wikidata SPARQL queries

cell2go.rq

Select Gene Ontology identifiers related to particular cell types

drugbank_info.rq

Acquire drugbank information from IDs

go2cell.rq

Select cells related to particular Gene Ontology identifiers

missing_description.rq

Find items without description

missing_refs_by_regex.rq

See articles with a main subject that have a particular regex on the title but no references for the main subject

number_of_statements.rq

Get the number of statements for each item

parkinsons_cells.rq

Cell types related to Parkinson's disease by markers

properties_of_kind.rq

Get all properties used for a kind on Wikidata

query_a_list_of_items.rq

Query Wikidata for information on a predefined set of items

string_search.rq

String search on a particular subset of items

wikidatagenes_for_goid.rq

Get Wikidata genes related to GO ID
